# 3DTiles Map Data Downloader

A Node.js tool for downloading 3DTiles map data, maintaining server directory structure and supporting resume functionality.

## Key Features

- ✨ Smart 3DTiles data structure parsing
- 📁 Automatic server directory structure replication
- ⏸️ Download resume capability
- 🔑 Automatic API key management
- ⚡ Multi-threaded concurrent downloads
- 🔄 Automatic recovery of interrupted downloads

## Getting Started

### System Requirements

- Node.js 14.0.0 or higher
- npm 6.0.0 or higher
